# Product Management API

A Spring Boot REST API for managing products with JWT authentication, role-based authorization, and Swagger documentation.

---

##  Tech Stack Used

- Java 21
- Spring Boot 3
- Spring Security + JWT
- MySQL 8
- Docker & Docker Compose
- Swagger / OpenAPI

---

## Features

- User authentication (JWT based)
- Role-based authorization (ROLE_ADMIN, ROLE_USER)
- Product CRUD APIs
- Pagination support
- Swagger API documentation
- Dockerized setup
